# 📚 Book Management System

A simple Full Stack Book Management System built using React.js, Axios, JSON Server, and Express-style backend deployment.

---

## 🚀 Live Demo

### Frontend (Vercel)
https://book-management-system-assignment-l.vercel.app/

### Backend (Render)
https://book-management-system-assignment-4.onrender.com

---

## 📌 Features

- View all books
- Add new books
- Edit existing books
- Delete books
- Genre filtering
- Pagination
- Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Axios
- Tailwind CSS
- React Icons

### Backend
- JSON Server
- Node.js
- Express-style setup
- Render Deployment
